Class Summary

In this class, we learned about a bunch of different topics, including:

  • Sensor use
  • Markov decision processes (MDPs)
  • Partially observable MDPs (POMDPs)
  • Kalman filters
  • Particle filters
  • SLAM and fastSLAM

Some of the algorithms learned were implemented in the lab. One of the assignments was to implement an interactive virtual particle filter, which I did using Processing. The code can be found on GitHub .
